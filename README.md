## 프로젝트 설명

- 호두랩스가 운영중인 땅콩스쿨 홈페이지의 메인 화면을 개발합니다.

## 프로젝트 실행 방법

1. git clone
   ```bash
   git clone
   ```
2. 패키지 설치
   ```bash
   npm install
   ```
3. local 프로젝트 실행
   ```bash
   npm run dev
   ```
